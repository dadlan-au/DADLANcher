# DADLANcher

Download the dadlancher.Bat, DADLAN-LOGO.ico and Asciilogo.txt file and put in a folder.
modify line 7 to reflect the location of the Asciilogo.txt dir
modify line 150 for the Quake3 install dir
Run dadlancher.bat

Font:

Layout:

Width: 92
Height: 70

Window Size:
